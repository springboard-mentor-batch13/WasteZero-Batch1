# WasteZero - Profile Logic Backend

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- bcrypt

---

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Go inside backend

```bash
cd backend
```

Install dependencies

```bash
npm install
```

Create .env

```
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/wastezero
JWT_SECRET=wastezero123
```

Run project

```bash
npm run dev
```

Server

```
http://localhost:5000
```
