# Team Setup Guide

## Requirements

Node.js

MongoDB

VS Code

---

## Install

```bash
npm install
```

---

## Create .env

```
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/wastezero
JWT_SECRET=wastezero123
```

---

## Run

```bash
npm run dev
```

---

## Backend URL

```
http://localhost:5000
```

---
